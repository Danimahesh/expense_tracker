# Expense Tracker

A premium offline Flutter expense tracking application with
smart billing-cycle support for credit cards and Amazon Pay Later.

## Features

- Offline SQLite database
- Material 3 UI
- Dark Mode
- Smart Billing Cycle Engine
- Interactive Charts
- PDF Export
- Excel Export
- Multiple Payment Methods
- Backup & Restore
